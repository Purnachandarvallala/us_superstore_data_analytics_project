# US Superstore Sales Analysis

 Project Overview
This project analyzes sales performance, profitability, and customer trends using the US Superstore dataset.

 Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook
- Power BI
- Excel

 Project Structure

data_analytics_project/
├── data/
│ └── US_Superstore.xlsx
├── notebooks/
│ └── 01_data_exploration.ipynb
├── dashboard/
│ └── Superstore_Dashboard.pbix
└── README.md


 Key Insights
- Technology category generates highest sales.
- West region contributes highest profit.
- High discounts reduce profit margins.

 Dashboard
Interactive dashboard built using Power BI.
